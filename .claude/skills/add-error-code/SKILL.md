# 에러코드 추가 규약

calc-harness에 새 에러코드를 추가할 때 아래 규칙을 반드시 지킨다.

## 번호 범위 (엄수)
- 1xxx = 입력/파싱 오류
- 2xxx = 수학 오류
- 9xxx = 예약 — 절대 사용 금지

## 절차
1. 새 에러가 어느 범주인지 판단한다 (입력이면 1xxx, 수학이면 2xxx).
2. 해당 범위에서 **이미 쓰인 번호를 확인하고, 그 다음 빈 번호**를 고른다.
   (예: 2xxx에 2001만 있으면 다음은 2002)
3. 9xxx 범위는 어떤 경우에도 쓰지 않는다.
4. 출력 형식은 반드시 `[E<코드>] <메시지>` 로 맞춘다.
5. 메시지는 한국어로, 사용자가 무엇이 잘못됐는지 알 수 있게 쓴다.
6. 에러 발생 시 stderr에 출력하고 exit(1)로 종료한다.

## 추가 후
- 추가한 코드와 의미를 파일 맨 위 주석의 범위 설명에 한 줄로 기록한다.
- 어떤 코드를 왜 추가했는지 사용자에게 한 문장으로 보고한다.
